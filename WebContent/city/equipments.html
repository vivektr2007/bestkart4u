TP-Link D-500

hardware features

  • Interface 4 10/100Mbps
  • LAN PORTS 1 10/100Mbps
  • WAN PORT Button
  • Quick Setup Security Button (WPS Compatible)
  • Reset Button
  • Power On/Off Button
  • External Power Supply 9VDC / 0.6A
  • Wireless Standards IEEE 802.11n*, IEEE 802.11g, IEEE 802.11b
